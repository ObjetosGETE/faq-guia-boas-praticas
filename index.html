FAQ - Guia de boas práticas para uso de tecnologias digitais

Olá, professor! Está com dúvidas sobre qual ferramenta da Microsoft utilizar em suas aulas? O Guia de boas práticas apresentou várias ferramentas da Microsoft. Nesta FAQ, você será direcionado à melhor opção de recurso para o que deseja realizar, seja para ação docente, seja para colaboração ou aula criativa.

Ação docente.

Eu gostaria de:

Criar uma planilha de cálculo para acompanhar o progresso dos alunos em atividades individuais.

O Excel oferece recursos avançados para criar e organizar dados de forma eficiente, sendo ideal para acompanhar o progresso dos alunos em atividades individuais.

Link tutorial - Treinamento em vídeo do Excel - Suporte da Microsoft

Criar documentos de texto, como trabalhos acadêmicos ou anotações de aula.

O Word é uma ferramenta amplamente utilizada para criar documentos de texto, proporcionando recursos avançados de formatação e edição para trabalhos acadêmicos e anotações de aula.

Link tutorial: Treinamento de Word para Windows - Suporte da Microsoft

Coletar feedbacks dos alunos sobre uma aula ou atividade.

O Forms é uma ferramenta eficaz para coletar feedbacks dos alunos de forma rápida e fácil, o que auxilia na melhoria contínua das práticas de ensino.

Link tutorial: Auxílio e aprendizado do Microsoft Forms

Armazenar e compartilhar documentos, apresentações e outros materiais educacionais de forma segura e acessível em nuvem.

O OneDrive é uma solução conveniente para armazenar e compartilhar materiais educacionais, proporcionando acesso fácil e seguro a partir de qualquer dispositivo.

Link tutorial: Treinamento em vídeo do OneDrive - Suporte da Microsoft

Aula criativa

Eu gostaria de:

Criar uma apresentação visualmente atraente e dinâmica.

O PowerPoint é uma ferramenta amplamente utilizada para criar apresentações visuais, que podem ajudar a explicar conceitos de forma envolvente durante as aulas.

Link tutorial: Treinamento de PowerPoint para Windows - Suporte da Microsoft

Criar apresentações ou relatórios digitais com um design dinâmico e interativo.

O Sway é uma ferramenta que permite criar apresentações e relatórios digitais com um design dinâmico e interativo, proporcionando uma experiência de aprendizado envolvente para os alunos.

Link tutorial: Introdução ao Sway - Suporte da Microsoft

Criar conteúdo educacional em formato de vídeo com recursos interativos.

O Flip é uma plataforma que permite criar vídeos educacionais interativos, o que pode aumentar o engajamento dos alunos durante o aprendizado.

Link tutorial: Flip no Microsoft Learn | Microsoft Learn

Colaboração

Eu gostaria de:

Facilitar a comunicação e colaboração entre professores e alunos.

O Teams oferece um ambiente completo para comunicação e colaboração entre professores e alunos, facilitando o trabalho em equipe e a troca de informações durante as aulas.

Link tutorial: MS-700 Introdução ao gerenciamento do Microsoft Teams - Training | Microsoft Learn

Facilitar a colaboração visual em tempo real durante as aulas.

O Whiteboard é uma ferramenta que permite a colaboração visual em tempo real durante as aulas, proporcionando um ambiente interativo para compartilhar ideias e resolver problemas.

Link tutorial: Introdução ao Microsoft Whiteboard - Suporte da Microsoft

Compartilhar vídeos educacionais.

O Stream é uma plataforma que permite hospedar e compartilhar vídeos educacionais, oferecendo uma maneira conveniente de acessar conteúdo de vídeo relacionado ao ensino.

Link tutorial: Como inserir vídeos do Stream no Loop e no Outlook - Suporte da Microsoft

Criar jogos virtuais em 3D.

O Minecraft Education foi criado para ajudar no desenvolvimento de atividades e proporcionar um aprendizado mais imersivo e divertido por meio da criação de mundos virtuais.

Link tutorial: Minecraft Education no Microsoft Learn | Microsoft Learn

Pesquisar recursos nas áreas de ciência, tecnologia, engenharia e matemática.

O STEM (Hacking STEM) oferece uma variedade de projetos práticos e experimentos científicos que podem ser realizados com equipamentos simples. Isso valoriza a criatividade, a inovação e as habilidades manuais, inserindo-se no contexto do movimento maker.

Para aprimorar seus conhecimentos, acesse o Guia de boas práticas para o uso de tecnologias digitais. Neste guia, que destaca as melhores práticas para aproveitar as ferramentas Microsoft 365, Minecraft Education e Hacking STEM, você encontrará diversas situações de aprendizagem para auxiliar no planejamento e engajamento de suas aulas. Além disso, convidamos você a contribuir para a construção deste documento compartilhando suas práticas educacionais.
